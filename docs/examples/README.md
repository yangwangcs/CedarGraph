# CedarGraph 事务模块示例

## 文件说明

- `transaction_examples.cc` - 完整的事务模块示例代码

## 编译运行

```bash
# 编译
g++ -std=c++17 -pthread transaction_examples.cc -o transaction_demo

# 运行
./transaction_demo
```

## 示例内容

### 示例1: 基础转账事务
演示最简单的ACID事务: 银行转账 (Alice -> Bob)

### 示例2: MVCC快照读取  
演示多版本并发控制: 事务T2在T3提交后仍能看到快照数据

### 示例3: OCC冲突检测
演示乐观并发控制: 两个事务并发修改同一数据，只有一个能成功

### 示例4: TWCD时间窗口冲突
演示CedarGraph特有的时间窗口冲突检测: 会议室预订系统

### 示例5: 批量事务
演示批量操作性能: 单事务插入1000条记录

### 示例6: 事务重试机制
演示带指数退避的事务重试: 10个线程并发转账

## 核心概念

- **OCC (乐观并发控制)**: 提交时验证冲突
- **MVCC (多版本并发控制)**: 读写不阻塞，支持快照
- **TWCD (时间窗口冲突检测)**: 检测时间范围重叠
