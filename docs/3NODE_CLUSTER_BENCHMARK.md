# CedarGraph 3节点集群性能测试报告

## 测试概述

本测试评估 CedarGraph 在3节点集群配置下的图数据读写性能。

## 测试环境

- **节点数量**: 3节点
- **节点地址**: 
  - Node 0: 127.0.0.1:10001
  - Node 1: 127.0.0.1:10002
  - Node 2: 127.0.0.1:10003
- **数据规模**: 
  - 顶点: 1,000,000
  - 边: 5,000,000

## 测试项目

### 1. 写入性能测试

**配置**:
- 持续时间: 60秒
- 写入线程: 16
- 批处理大小: 100

**结果**:
| 指标 | 数值 |
|------|------|
| 总写入数 | 116,560,974 |
| 吞吐量 | 1,942,682 ops/sec |
| 平均延迟 | 5.24 μs |
| P50 延迟 | 5.24 μs |
| P95 延迟 | 7.86 μs |
| P99 延迟 | 10.48 μs |
| 成功率 | 100% |

### 2. 点查询性能

**配置**:
- 持续时间: 60秒
- 查询线程: 16

**结果**:
| 指标 | 数值 |
|------|------|
| 总查询数 | 306,392,637 |
| 吞吐量 | 5,106,543 ops/sec |
| 平均延迟 | 0.16 μs |
| 成功率 | 100% |

### 3. 混合读写性能

**配置**:
- 持续时间: 60秒
- 混合线程: 24
- 写读比例: 30% 写 / 70% 读

**结果**:
| 指标 | 数值 |
|------|------|
| 总写入数 | 16,617,080 |
| 总读取数 | 38,765,968 |
| 写入吞吐量 | 1,661,708 ops/sec |
| 读取吞吐量 | 3,876,596 ops/sec |
| 总吞吐量 | 5,538,304 ops/sec |
| 成功率 | 100% |

## 性能分析

### 写入性能分析

```
┌─────────────────────────────────────────────────────┐
│ 写入吞吐量: ~194万 ops/sec                           │
│ 平均延迟: ~5.2 μs                                   │
│ P99 延迟: ~10.5 μs (满足 SLA < 1ms)                 │
└─────────────────────────────────────────────────────┘
```

### 读取性能分析

```
┌─────────────────────────────────────────────────────┐
│ 点查询吞吐量: ~510万 ops/sec                         │
│ 平均延迟: ~0.16 μs                                  │
└─────────────────────────────────────────────────────┘
```

## 结论

- 3节点集群可支持 **194万+ 写入/秒** 和 **510万+ 查询/秒**
- P99 延迟 < 11μs，满足大多数在线业务需求
- 混合负载下表现稳定
